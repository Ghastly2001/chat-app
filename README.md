#  Chatify — Real-Time Chat App (Next.js + Firebase)

A fully-featured real-time chat application built with **Next.js 13** and **Firebase v9**, delivering a seamless, WhatsApp-like experience.  
Supports **1:1 chat**, **group chat**, **media sharing**, **typing indicators**, **blue ticks**, and **message receipts**.  

---

##  Features

###  Real-Time Messaging
- Live chat powered by Firebase Firestore
- Instant updates without page reload

###  WhatsApp-Style UX
- **Double ticks** → Delivered  
- **Blue ticks** → Seen  
- **Typing indicator**  
- **Last seen**  
- Smooth message bubble design

###  Multimedia Support
- Image upload & preview  
- Full-screen image viewer  
- Emoji picker  
- Click-away listener for dropdowns

###  Authentication
- Firebase Authentication  
- Secure session handling

###  Message Management
- Read receipts  
- Seen/unseen logic  
- Auto-scroll to last message  
- Timestamps with Moment.js (react-moment)

###  Responsive UI
- Built with Tailwind CSS  
- Mobile-first layout  
- Desktop split-view layout

---

##  Tech Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | Next.js 13, React 18 |
| **Backend** | Firebase (Auth, Firestore, Storage) |
| **UI** | Tailwind CSS, React Icons |
